## 🚀 Funcionalidades Principais

- **🧩 Divisão em Páginas**: Transforme uma única imagem em um poster de até 6 páginas (A4).
- **🎨 Filtros Profissionais**: Ajuste nitidez, contraste e saturação em tempo real.
- **📐 Controle de Layout**: Alterne entre orientação vertical/horizontal e ajuste margens em milímetros.
- **🔍 Edição por Página**: Reposicione o crop de cada página individualmente para um ajuste perfeito.
- **📦 Exportação Flexível**: Gere seu poster em **JPG**, **PNG** ou **PDF** e baixe tudo compactado em um arquivo **.ZIP**.
- **🔒 Privacidade Total**: Todo o processamento ocorre localmente no seu navegador. Nenhuma imagem é enviada para servidores.

## 🛠️ Como Funciona

1.  **Upload**: Envie sua arte nos formatos JPG ou PNG.
2.  **Ajustes**: Utilize o painel lateral para aplicar filtros, definir o número de páginas, margens e orientação.
3.  **Refinamento**: Se necessário, ajuste o enquadramento de cada página individualmente arrastando a imagem no editor de páginas.
4.  **Exportação**: Clique em "Confirmar e gerar poster", escolha o formato desejado e receba seu arquivo pronto para impressão.

## 🧰 Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias modernas de front-end:

- [Tailwind CSS](https://tailwindcss.com/): Para uma interface moderna, responsiva e com suporte a temas.
- [SweetAlert2](https://sweetalert2.github.io/): Para feedbacks e popups elegantes.
- [JSZip](https://stuk.github.io/jszip/): Para a geração do arquivo compactado final.
- [jsPDF](https://rawgit.com/MrRio/jsPDF/master/docs/index.html): Para a geração de documentos PDF de alta qualidade.
- **Canvas API**: Para todo o processamento e manipulação de imagens.

## ❓ FAQ (Perguntas Frequentes)

**O app envia minha imagem para o servidor?**
Não. O processamento ocorre 100% localmente no seu navegador, garantindo sua privacidade.

**Quantas páginas posso gerar?**
Atualmente, o sistema suporta a divisão em até 6 páginas.

**Posso corrigir o enquadramento de uma página específica?**
Sim! O editor permite selecionar uma página individual e arrastar a imagem para ajustar o corte exatamente como você deseja.
